# web-bundling

> Bundle web applications into single HTML files for distribution. Use when creating self-contained HTML games, artifacts, or demos that need to be shared as a single file. Covers React+Vite+Parcel bundling pattern.

---

## Quick Start

### Prerequisites

- OpenClaw gateway running (`openclaw gateway status`)
- Workspace with skills directory

### Installation

Part of the OpenClaw skills collection — no separate install needed.

```bash
ls skills/web-bundling/SKILL.md
```
