## Key Technical Lessons
- Never use `echo password | sudo -S nohup cmd &` over SSH — pipe + background breaks stdin
- Even Paramiko `stdin.write(password)` + `nohup &` doesn't fully detach — need `bash -c` wrapper with `</dev/null`
- Wi-Fi daemon labels differ from docs: `com.apple.wifi.WiFiAgent` not `com.apple.WiFiAgent`, `com.apple.mDNSResponder.reloaded` not `com.apple.mDNSResponder`
- `skip_snapshot: true` needed — tmutil fails with exit 73 on this target
